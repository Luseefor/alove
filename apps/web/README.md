# apps/web

Next.js app for the LaTeX editor and Convex backend client.

Setup and run instructions live in the [repository root README](../../README.md). Quick reference:

```bash
# from monorepo root
bun install
bun run dev
# editor: URL printed in terminal (local scan from ALOVE_WEB_PORT, default 30127)
# cloud stack: see repository README (dev:cloud + convex:dev)
```
